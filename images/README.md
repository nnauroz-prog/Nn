# Bilder für Villa Dream

Damit die Website lebt, müssen die folgenden Bilddateien hier im Ordner liegen.
**Die Dateinamen müssen exakt stimmen** (Kleinschreibung, Bindestrich, Endung `.jpg`).

## Erforderliche Dateien

| Dateiname           | Verwendung                                           | Empfohlene Größe         |
|---------------------|------------------------------------------------------|--------------------------|
| `saal-1.jpg`        | Kapitel II („Die Tafel") · Galerie 01                | quer, ~2400×1800 px      |
| `saal-2.jpg`        | **Hero-Bild (Startseite)** · Kapitel I · Galerie 02  | quer, ~2400×1600 px      |
| `saal-3.jpg`        | Kapitel III („Die Trauung") · Galerie 03 · CTA       | quer, ~2400×1600 px      |
| `mohammad.jpg`      | Über-uns-Sektion (optional)                          | hoch, ~1200×1500 px      |

## So lädst du die Bilder hoch

### Über GitHub im Browser
1. Geh auf https://github.com/nnauroz-prog/Nn/tree/main/images
2. Oben rechts „Add file" → „Upload files"
3. Bilder reinziehen, **vorher unbedingt umbenennen** (`saal-1.jpg`, `saal-2.jpg`, `saal-3.jpg`)
4. Unten „Commit changes"

### Über Drag & Drop in Claude
Lade sie einfach in unseren Chat hoch — ich speichere sie unter den richtigen Namen ins Repo.

## Format-Hinweise

- **JPG** bevorzugt (kleiner als PNG, ausreichend für Fotos)
- **Querformat** für Saal-Bilder, **Hochformat** für Personenfoto
- **Maximale Dateigröße**: 2 MB pro Bild (sonst lädt die Seite langsam)
- Falls deine Bilder größer sind, online komprimieren auf https://squoosh.app

Solange die Dateien fehlen, zeigt die Website elegante Marmor-Verläufe als Platzhalter.
